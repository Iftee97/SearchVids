
To make and push changes to github repo using git:
--------------------------------------------------

  git clone https://github.com/Iftee97/SearchVids.git
  
  cd SearchVids
  
  git checkout -b branchName {optional}

  git add .
 
  git commit -m ""

  git push OR, git push origin branchName
